# angularjs-web-planet-weight

## Description
Calculates a weight, assumed in pounds,
for each of the eight planets.

## Tech stack
- angular 1.6

## Docker stack
- httpd:latest

## To run
`sudo ./install.sh -u`  
http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
